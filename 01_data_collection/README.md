# 01 — Data Collection

Gathering parallel corpora and audio data for Kabyle NLP tasks.

## Contents

| File | Description |
|------|-------------|
| `scrape_radio_algerie_podcasts.ipynb` | Scrapes Kabyle audio extracts from Radio Algérie podcast platform |
| `tatoeba_kabyle_audio.csv` | Metadata for Kabyle audio sentences from Tatoeba |
| `download_data.ipynb` | Downloads Kabyle data from MADLAD-400 (HuggingFace) |

## Source

- **MADLAD-400**: Google's large web-crawled parallel corpus
- **Radio Algérie**: Podcast platform with Kabyle (Chaine 2) content
- **Tatoeba**: Crowd-sourced sentence translations with audio
